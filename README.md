# shug 的笔记库

这是一个基于 Starlight 的中文笔记站点，内容按主题合集整理，适合直接浏览或本地维护。

## 常用命令

- `npm install`：安装项目依赖，首次克隆后先跑这个命令。
- `npm run dev`：启动本地开发站点，适合边改边看。
- `npm run build`：生成正式构建产物，检查站点能否成功打包。
- `npm run docs:check`：执行文档检查流程，包含格式、拼写、链接和构建验证。

## 写作规范

提交或修改文档前，先看 [DOCS_STYLE.md](DOCS_STYLE.md)。里面说明了中文优先写法、Starlight 路由链接规则和参考资料要求。
